#  Private Transcriber

**A voice transcriber that runs fully on-device with a latency of ~100ms.**
**Private Alternative to WhisperFlow. Faster Than VoiceBox. **
**No 4GB of model download. No 5 seconds of waiting after a 5 seconds dictation.**


## Installation

You can either download the entire repository and run/build it with Xcode or you can download the latest signed `dmg` for the app from the [GitHub release page](https://github.com/0Itsuki0/PrivateTranscriber/releases).


## Features

- Custom Global Hotkey
- Activation mode: hold-to-speak or tap-to-start/stop
- Auto Language Detection
- Optional LLM Formatting
